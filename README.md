# Portfolio

Personal portfolio website showcasing my work as a DevOps Engineer.

## About

This portfolio highlights my experience with infrastructure automation, containerization, and network operations. Built with modern web technologies to demonstrate both technical capability and design sensibility.

## Tech Stack

- Next.js 16
- React with TypeScript
- Tailwind CSS
- Three.js for WebGL effects
- Font Awesome icons

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```
